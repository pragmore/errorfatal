Clon de StackOverflow

## Frontend

El frontend es una app hecha en [Next.js](https://nextjs.org/).

## Setup

Instalar Node y yarn 

```bash
cd front
yarn install 
```

## Correr

Ir a la carpeta `front`. y ejecutar

```bash
yarn dev
```

Ir a [http://localhost:3000](http://localhost:3000).
